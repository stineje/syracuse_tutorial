# setup_opencircuitdesign.sh

This script provides a convenient environment setup for the Open Circuit Design open-source VLSI toolchain installed under your local user directory.

It is intended for users working with tools such as:

- Magic (layout editor and extraction)
- Netgen (LVS verification)
- ngspice (circuit simulation)
- Xschem (schematic capture)
- IRSIM (switch-level simulation)
- open_pdks (PDK installation and management)

The default installation location assumed by this script is:

```bash
$HOME/opencircuitdesign
